
# TuBuBuApp

TuBu 是一个面向徒步、露营和轻量户外出行的微信小程序，用于管理个人装备库、规划出行方案、统计装备重量，并完成打包检查。

当前版本为 V1.0，主要围绕个人本地使用场景，不包含社区、商城、会员、后端服务或真实 AI 功能。

## 功能

- 装备库管理：新增、编辑、删除和查看个人装备
- 出行方案管理：创建、编辑、复制和设置默认方案
- 方案清单：将装备加入指定方案，保存装备快照
- 重量统计：按方案统计总重量和分类重量
- 打包检查：标记方案装备是否已经打包
- 首页概览：展示装备库、默认方案和当前方案统计
- 本地存储：使用微信小程序本地存储保存数据

## 本地运行

1. 克隆仓库：

```bash
git clone https://github.com/Pupu727/TuBuBuApp.git
```

2. 使用微信开发者工具打开项目根目录。

3. 确认小程序目录为：

```txt
miniprogram/
```

4. 在微信开发者工具中编译运行。

## 数据说明

V1.0 默认使用本地存储，不依赖后端服务。

核心数据包括：

- `Gear`：装备库中的装备
- `TripPlan`：出行方案
- `PlanItem`：方案中的装备快照

其中 `PlanItem` 会保存装备名称、分类、重量和价格快照，因此后续修改装备库不会影响历史方案。
