# 03 Stage Gear Library

## goal

实现装备库管理：新增、编辑、删除、查看、搜索、分类筛选和基础统计。

## scope

- 装备列表页展示装备卡片。
- 支持分类筛选和名称搜索。
- 支持新增、编辑、软删除装备。
- 支持装备图片字段和本地临时图片预览。
- 展示持有天数和日均成本。

## data impact

- 写入和读取 `Gear` 本地存储。
- 删除使用软删除，不物理移除历史数据。
- 购买日期和价格存在时计算日均成本。
- 装备状态支持：using、idle、wishlist、broken、borrowed。

## UI impact

- 装备卡片展示名称、分类、重量、价格、状态、持有天数、日均成本。
- 装备表单包含名称、分类、重量、价格、状态、购买日期、渠道、备注、图片。
- 装备名称和重量必填；价格可为空。
- 空列表和搜索无结果必须有提示。

## acceptance criteria

- 可录入“艾王 35+5L / 背负系统 / 1100g / 125元 / 使用中”。
- 录入后装备统计显示数量 1、总重量 1.1kg、总价值 125元。
- 可按“背负系统”筛选看到该装备。
- 可搜索“艾王”找到该装备。
- 删除装备后列表不显示，但历史方案快照不受影响。

## out of scope

- 从装备库加入方案。
- 方案详情。
- 批量智能添加。
- 云端图片上传。
