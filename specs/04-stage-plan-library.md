# 04 Stage Plan Library

## goal

实现出行方案管理：新建、编辑、删除、复制、设置默认和方案列表统计。

## scope

- 方案列表页展示全部未删除方案。
- 支持新建、编辑、软删除方案。
- 支持复制历史方案。
- 支持设置当前默认方案。
- 方案卡片展示装备数量、总重量和打包进度。

## data impact

- 写入和读取 `TripPlan` 本地存储。
- 复制方案时复制方案基本信息和 `PlanItem` 快照。
- 复制后的打包状态重置为 `unpacked`。
- 同一时间只允许一个默认方案。

## UI impact

- 方案表单包含名称、路线/地点、出发日期、结束日期、天数、出行类型、天气备注、目标重量、备注。
- 方案名称必填。
- 删除方案前必须二次确认。
- 默认方案在列表中有醒目标识。

## acceptance criteria

- 可创建“冬五台山重装”方案。
- 可设置该方案为默认方案。
- 可编辑路线和出发日期。
- 可复制该方案并生成新的方案记录。
- 删除方案后列表不再显示。

## out of scope

- 方案装备清单编辑。
- 从装备库添加装备。
- 重量图表。
- 云端同步。
